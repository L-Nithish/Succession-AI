AI-Powered Interview Preparation Ecosystem Walkthrough

Technical Stack Summary

Backend (Spring Boot 3 + Java 21)

  • Stateless Security: JWT filters with token rotation.
  • WebSocket Brokers: Live SockJS chat channels with typing triggers.
  • Local Compiler Sandbox: Curly braces checker emitting execution logs.
  • AI Fallback: Fully offline keyword scoring engine.

Frontend (Next.js 14 + Tailwind CSS)

  • Cinematic Theme: Deep blacks, glowing borders, and glass cards.
  • Smooth Scrolling: Lenis RAF loop scrolling implementation.
  • 6 Routes: Home Landing, Features Analyzer, Mock Sandbox Room, Analytics Stats, Recruiter Rankings, Pricing Tiers.

Verification Scores

  • Backend Compilation: mvn clean compile runs with BUILD SUCCESS (39 source files packaged).
  • Frontend Compilation: npm run build builds cleanly with ✓ Compiled successfully (9 static routes compiled).

Launch Instructions

Since Docker is not configured on the local host path, you can run the entire database-connected stack directly on Windows:

Step 1: PostgreSQL Setup

Ensure PostgreSQL is listening on port 5432 with username postgres and password postgres, and that database interviewacedb exists.

Step 2: Start Spring Boot Backend

C:\Users\dexte\.m2\wrapper\dists\apache-maven-3.9.16-bin\5grr65jo27hi51sujmtcldfovl\apache-maven-3.9.16\bin\mvn.cmd spring-boot:run

Step 3: Start Next.js Frontend

npm.cmd run dev

Once started, access the interfaces:

  • Frontend Web App: http://localhost:3000
  • Backend REST API: http://localhost:8080/api/auth/register
